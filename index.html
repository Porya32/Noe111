🌐 زمین بازی Linux

🔹 سیستم‌عامل‌ها: Linux، Unix، iOS

لینوکس یک سیستم‌عامل آزاد و متن‌باز است که بر پایه یونیکس طراحی شده است. یونیکس، پایه‌گذار بسیاری از مفاهیم سیستم‌عامل مدرن است. iOS سیستم‌عاملی است که اپل برای دستگاه‌های موبایل خودش توسعه داده و از پایه‌های یونیکس بهره می‌برد.

💾 سرور چیست؟

سرور یک سیستم کامپیوتری است که خدماتی مانند ذخیره‌سازی، ارسال داده، اجرای نرم‌افزار و مدیریت پایگاه‌داده را به سایر کامپیوترها ارائه می‌دهد. لینوکس معمولاً سیستم‌عامل اصلی در اکثر سرورهاست.

📦 توزیع‌های معروف لینوکس

  • Ubuntu
  • Debian
  • Arch Linux
  • Fedora
  • Kali Linux (برای تست نفوذ)

💻 دستورات کاربردی لینوکس

  • ls → نمایش لیست فایل‌ها
  • cd → تغییر مسیر
  • mkdir → ساخت پوشه
  • rm → حذف فایل/پوشه
  • chmod → تغییر سطح دسترسی
  • sudo → اجرای دستور با دسترسی ادمین
  • apt update && apt upgrade → بروزرسانی
